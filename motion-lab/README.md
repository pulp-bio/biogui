# MotionLab

## Project Setup & First Steps

1.  **Clone the Repository:**

    ```bash
    git clone <REPO_URL>
    ```

2.  **Add the Project in Unity Hub:**
    Start the **Unity Hub** and add the cloned repo from disk. Unity will prompt you to install the correct editor version if not installed.
    Unity will import the project and build the local `Library` folder.

3.  **Open the Project in Unity:**
    Launch the project in the **Unity Hub**. The first time, unity will download packages in the background.

4.  **Run the Project:**
    Once launched, you have to open one of the unity scenes:
    - `Assets/_Project/Scenes/ContinuousTasksScene.unity` or
    - `Assets/_Project/Scenes/MainScene.unity`
      You can run the scene by pressing the play button on top.
